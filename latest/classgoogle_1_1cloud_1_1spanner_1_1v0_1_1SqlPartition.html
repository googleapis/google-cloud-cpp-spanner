Detailed Description

The SqlPartition class is a regular type that represents a single slice of a parallel SQL read.

Instances of SqlPartition are created by Client::PartitionSql. Once created, SqlPartition objects can be serialized, transmitted to separate process, and used to read data in parallel using Client::ExecuteSql.

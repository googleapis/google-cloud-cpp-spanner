Detailed Description

A struct that represents a collection of bytes.

This struct is a thin wrapper around a std::string to distinguish BYTES from STRINGs. The .data member can be set/get directly. Consructors and relational operators are provided for convenience and to make Bytes a regular type that is easy to work with.

Note that the relational operators behave as if the std::string chars are unsigned, which is exactly how Spanner BYTES values compare.

Detailed Description

The KeySet class is a regular type that represents the collection of spanner::Rows necessary to uniquely identify a arbitrary group of rows in its index, or all the keys in an index.

Unlike KeySetBuilder, KeySet stores its keys and key ranges in a type erased fashion.
